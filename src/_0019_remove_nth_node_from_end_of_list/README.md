# [19. Remove Nth Node From End of List][title]

## Description

Given the `head` of a linked list, remove the n<sup>th</sup> node from the end of the list and return its head.

**Example 1:**
```text
Input: head = [1,2,3,4,5], n = 2
Output: [1,2,3,5]
```

**Example 2:**
```text
Input: head = [1], n = 1
Output: []
```

**Example 3:**
```text
Input: head = [1,2], n = 1
Output: [1]
```
