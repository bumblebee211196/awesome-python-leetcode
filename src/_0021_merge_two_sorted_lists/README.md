# [21. Merge Two Sorted Lists][title]

## Description

Merge two sorted linked lists and return it as a **sorted** list. The list should be made by splicing together the nodes of the first two lists.

**Example 1:**
```text
Input: l1 = [1,2,4], l2 = [1,3,4]
Output: [1,1,2,3,4,4]
```

**Example 2:**
```text
Input: l1 = [], l2 = []
Output: []
```

**Example 3:**
```text
Input: l1 = [], l2 = [0]
Output: [0]
```
