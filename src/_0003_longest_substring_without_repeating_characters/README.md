# [3. Longest Substring Without Repeating Characters][title]

## Description

Given a string `s`, find the length of the **longest substring** without repeating characters.

**Example 1:**
```text
Input: s = "abcabcbb"
Output: 3
```

**Example 2:**
```text
Input: s = "bbbbb"
Output: 1
```

**Example 1:**
```text
Input: s = "pwwkew"
Output: 3
```

**Example 2:**
```text
Input: s = ""
Output: 0
```
